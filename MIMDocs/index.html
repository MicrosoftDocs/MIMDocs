Microsoft Identity Manager

Microsoft Identity Manager provides on-premises identity and access management and enables you to:

  • Synchronize identities between directories, databases and applications
  • Set up self-service password, group, and certificate management
  • Increase admin security with policies, privileged access and roles
  • Thwart identity theft
